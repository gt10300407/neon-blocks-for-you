# 유진공주 테트리스 v3.5

드롭 점수를 일반적인 현대 테트리스 방식에 가깝게 낮춘 버전이다.

## v3.5 변경

- 자연 낙하 후 착지: 0점
- 소프트 드롭: 내려간 칸당 1점
- 하드 드롭: 내려간 칸당 2점
- 줄 삭제, 콤보, 백투백, 퍼펙트 클리어 점수는 기존과 동일
- NEON OVERDRIVE와 특수 도형 시스템 유지
- 기존 브라우저 최고점수 기록 유지

점수의 중심을 단순 낙하가 아니라 줄 삭제와 콤보에 두도록 조정했다.
