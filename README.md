# 유진공주 테트리스 v3.8

스와이프 방향 오인식을 수정한 버전이다.

## v3.8 변경

- 스와이프 방향 잠금 추가
- 처음 약 12px 이동 후 가로/세로 방향 판정
- 세로 스와이프에는 좌우 흔들림을 넉넉하게 허용
- 세로 방향으로 잠긴 뒤 좌우 이동 완전 무시
- 가로 방향으로 잠긴 뒤 위아래 이동 완전 무시
- 애매한 대각선 입력은 방향이 분명해질 때까지 대기
- 탭 회전, 위 스와이프 홀드, 빠른 아래 플릭 하드 드롭 유지
- v3.7의 25초 AUTO FEVER, 최고기록, 18종 도형 유지

이제 아래로 내리는 중 손가락이 약간 좌우로 흔들려도 블록이 옆으로 이동하지 않는다.
