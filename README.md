# 유진공주 테트리스 v3.6

모바일 조작 피로, 최고기록 표시, 도형 다양성을 개선한 버전이다.

## v3.6 핵심 변경

### 스와이프 전용 조작

게임 조작용 버튼을 전부 제거했다.

- 좌우 스와이프: 블록 여러 칸 이동
- 짧은 탭: 회전
- 아래 스와이프: 소프트 드롭
- 빠른 아래 플릭: 하드 드롭
- 위 스와이프: 홀드
- 스와이프 감도: 편안함 / 보통 / 빠름

### AUTO FEVER

- 게이지가 100%가 되면 자동 발동
- 수동 DRIVE 버튼과 V 키 제거
- 8초간 점수 2배
- 낙하 속도 완화
- 콤보 보호 2회

### 최고기록

- 상단 HUD에 BEST 상시 표시
- 게임 종료 시 이번 점수, 최고점수, 라인, 레벨, 최대 콤보 표시
- NEW RECORD 연출
- 현재 브라우저에 최고기록 TOP 5 저장

### 도형 18종

- 기본 도형 7종
- 특수 도형 11종
- 레벨 1~2: 초기 특수 도형 4종
- 레벨 3~5: 특수 도형 4종 추가
- 레벨 6 이상: 전체 18종 등장
- 특수 도형 확률: 15% → 20% → 25%
