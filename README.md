# 유진공주 테트리스 v3.7

AUTO FEVER 지속시간과 연장 규칙을 개선한 버전이다.

## v3.7 변경

- AUTO FEVER 기본 지속시간: 8초 → 25초
- 피버 중 줄 삭제: 한 줄당 3초 연장
  - 1줄: +3초
  - 2줄: +6초
  - 3줄: +9초
  - 4줄: +12초
- 피버 최대 지속시간: 40초
- 일시정지 중에는 피버 시간이 흐르지 않음
- 피버 중 줄 삭제 문구에 연장 시간을 표시
- 기존 점수 2배, 낙하 속도 완화, 콤보 보호는 유지

레벨 1에서 자연 낙하만 사용할 경우 한 블록이 바닥까지 이동하는 데 최대 약 18초가 걸리므로,
기본 25초로 조정해 최소 한 블록 이상은 피버 효과를 체감할 수 있게 했다.
