# 유진공주 테트리스 v3.10

v3.9에서 제거했던 플레이 중 조작 안내 글자와 화살표를 다시 표시하고,
iPhone Safari의 길게 누르기 확대경만 차단한 버전이다.

## v3.10 변경

- 조작 패드의 `← →` 화살표 다시 표시
- `SWIPE TO MOVE`
- `TAP ROTATE · ↑ HOLD · ↓ DROP`
- 화면상 글자는 원래와 동일하게 복구
- 안내는 선택 가능한 HTML 텍스트가 아니라 CSS 생성 문구로 표시
- iOS가 길게 누르기 텍스트 확대경을 잡을 실제 선택 대상 제거
- `touchstart`, `touchmove`, `touchend` 기본 동작 차단 강화
- 스와이프 방향 잠금, 25초 AUTO FEVER, 기록, 도형 18종 유지
