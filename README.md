# 유진공주 테트리스 v3.4

- 브라우저 탭 제목: 유진공주 테트리스
- 상단 게임 로고: 유진공주 테트리스
- 시작 화면 제목: 유진공주 테트리스
- 접근성 게임 이름과 설명 변경
- 게임 규칙, 점수, 난이도, 저장된 최고점수는 그대로 유지
- v3.3의 클래식 7종 + 특수 도형 5종 및 History Randomizer 유지
